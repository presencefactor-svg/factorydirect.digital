# factorydirect.digital

Static Cloudflare Pages repository for Factory Direct Multimedia.

## Routes

```text
/                       Restored Factory Direct homepage with Woo Plugins link
/privacy-policy.html    Privacy policy from archived site
/woo/                   Customer-facing multilingual WooCommerce marketplace connector landing
/woo/documentation/     Multilingual installation and configuration guide
```

PRO setup buttons point to the contact section until final payment links are supplied.
