Marketplace connector suite

Two regional connectors, one operational workflow.

Each plugin follows the same WooCommerce-first approach, while respecting the rules and API behavior of the local OLX marketplace.

OLX.ba

WooCommerce Connector for OLX.ba

Designed for Bosnia & Herzegovina sellers who need controlled product sync, safe listing metadata preservation and category/attribute mapping for OLX.ba workflows.

  • ✓WooCommerce product to OLX.ba advert sync
  • ✓Category and required attribute support
  • ✓Safe updates without duplicate listings
OLX.bg

Marketplace Connector BG for WooCommerce

A Bulgarian-market connector using OLX.bg Partner API v2, OAuth connection, category suggestions and per-category attribute payloads.

  • ✓OAuth 2.0 and Bearer API requests
  • ✓Create/update adverts with required state attribute
  • ✓Lite package plus PRO automation path
