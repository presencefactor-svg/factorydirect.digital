Two regional plugins, one WooCommerce workflow.

Each connector is tailored to the local OLX platform: API method, category/attribute rules, account requirements and moderation behavior.

OLX.ba

WooCommerce Connector for OLX.ba

Designed for Bosnian OLX workflows, category mapping, required attributes and safe preservation of existing listing metadata.

  • ✓WooCommerce product to OLX.ba listing synchronization
  • ✓Category and required attribute handling
  • ✓Safe update flow that avoids duplicate listings
OLX.bg

Marketplace Connector BG for WooCommerce

A Bulgarian-market connector using the OLX.bg Partner API v2, OAuth connection, category suggestions and per-category attributes.

  • ✓OAuth 2.0 connection and Bearer API requests
  • ✓Create/update adverts with required state attribute
  • ✓Lite for WordPress.org, PRO for automation
